theWorldIn35mm Theme for WordPress
-------------------------

* You can view in action for wordpress theWorldIn35mm at: 
  http://www.etruscosphoto.it

* You can download newest releases at:
  http://www.pensando.it/wp/index.php/template



1) Installation prerequisites

* Get and install YAPB (http://johannes.jarolim.com/yapb)
  The "theWorldIn35mm" theme won't work without the YAPB plugin. 
  You will find installation  instructions on the YAPB website.


2) Installation instructions

* Extract the archives's contents into the wp-content/themes 
  directory of your WordPress installation.


3) Configuration

* See http://www.pensando.it/wp/index.php/template for more information about the configuration options.

4) Have fun :-)
